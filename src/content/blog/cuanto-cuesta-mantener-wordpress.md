## Desglose detallado

### 1. Hosting (€8-30/mes)

El hosting es donde vive tu web. **No escatimes aquí.**

**Hosting compartido (€8-12/mes)**
- Para: Webs pequeñas, blogs, <5.000 visitas/mes
- Ejemplos: Webempresa Basic, SiteGround StartUp
- ✅ Barato
- ❌ Más lento, recursos limitados

**VPS/Cloud (€15-30/mes)**
- Para: Webs profesionales, e-commerce, >10.000 visitas/mes
- Ejemplos: DigitalOcean, Vultr, Webempresa PRO
- ✅ Rápido, escalable
- ❌ Más caro, puede requerir conocimientos técnicos

**WordPress gestionado (€20-80/mes)**
- Para: Quien quiere olvidarse de todo
- Ejemplos: Kinsta, WP Engine, Flywheel
- ✅ Optimizado para WordPress, soporte experto, backups incluidos
- ❌ Más caro

**Mi recomendación:**
- **Proyecto nuevo:** Webempresa o SiteGround (€10-12/mes)
- **E-commerce:** DigitalOcean + hosting gestionado por desarrollador (€20-25/mes)

### 2. Dominio (€10-15/año)

Tu dirección web (ejemplo.com). Coste bastante fijo.

**Consejos:**
- Compra el dominio SEPARADO del hosting (más control)
- Proveedores: Namecheap, Google Domains, Hostinger
- **Renovación automática:** Actívala o perderás el dominio

### 3. Certificado SSL (€0)

Antes costaba dinero, ahora es gratis con Let's Encrypt (incluido en todos los hostings modernos).

**Obligatorio en 2026:** Google penaliza webs sin HTTPS.

### 4. Plugins premium (€0-500/año)

Depende de tus necesidades. Mis recomendaciones:

**Gratis (suficiente para mayoría):**
- SEO: Yoast SEO Free
- Backups: UpdraftPlus Free
- Seguridad: Wordfence Free
- Caché: WP Fastest Cache Free

**Premium solo si necesitas:**
- **Elementor Pro** (€59/año): Constructor visual avanzado
- **ACF Pro** (€49/año): Campos personalizados complejos
- **WooCommerce extensions** (€50-200/año): Funcionalidades e-commerce específicas
- **WPML** (€99/año): Webs multiidioma complejas

### 5. Tema premium (€0-300)

**Opciones:**

**Gratis:**
- Astra
- GeneratePress
- Kadence

Suficiente para 80% de proyectos.

**Premium:**
- Astra Pro: €47/año
- GeneratePress Premium: €49/año
- Divi: €89/año

**Custom (desarrollador):** €800-2.000 único (luego gratis)

### 6. Mantenimiento (€10-150/mes)

**El más importante y más olvidado.**

#### Opción A: Tú mismo (€10/mes = tu tiempo)

**Tareas mensuales necesarias:**
- Actualizar WordPress core (5 min)
- Actualizar plugins (10 min)
- Actualizar tema (2 min)
- Revisar backups funcionan (10 min)
- Revisar seguridad (escaneo) (5 min)
- Optimizar base de datos (5 min)
- Revisar enlaces rotos (10 min)

**Total:** ~1 hora/mes

**¿Puedes hacerlo?** Sí, si:
- ✅ Eres medianamente técnico
- ✅ Tienes tiempo cada mes
- ✅ Tu web no es crítica para tu negocio

#### Opción B: Freelance/Agencia (€40-150/mes)

**Qué incluye un buen servicio:**
- Actualizaciones (core, plugins, tema)
- Backups automáticos
- Monitoreo uptime
- Escaneos de seguridad
- Optimizaciones de rendimiento
- Informes mensuales
- Soporte para dudas

**Precio justo:** €40-80/mes para web normal, €100-150/mes para e-commerce

## Errores comunes que cuestan dinero

1. **Hosting muy barato** → Web lenta → Pierdes visitas
2. **No hacer mantenimiento** → Hackeo → €500-1.000 en limpieza
3. **Plugins pirateados** → Malware garantizado
4. **No tener backups** → Se rompe algo → Pierdes la web
5. **Ignorar actualizaciones** → Vulnerabilidades de seguridad

## ¿Vale la pena mantener WordPress?

**SÍ, si:**
- Tu web genera ingresos o leads
- Tiene buen tráfico
- Es profesional para tu negocio

**Considera cambiar a alternativa si:**
- Solo necesitas web informativa estática
- No tienes tiempo ni presupuesto para mantenerla
- Nunca actualizas nada

**Alternativa:** Sitios estáticos (Astro, Next.js) - Casi cero mantenimiento
